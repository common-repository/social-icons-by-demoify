==Description==
Social-Icons-By-Demoify provides a user friendly interface to link with the social networking sites. It user latest and most popular social media icons for users. Users can add this plugin to widgets and can show where ever necessary and can manually insert all the links to the social icons.
One can also change different styles to display the icons. Multiple options are provided for user benefits. We also provide dedicate support to user regarding this plugin. We are keeping the plugin small and light weight so that the site would not be affected in its performance.
There are several social icons available:

*Facebook
*Google +
*Twitter
*Instagram
*Pininterest
*Git
*Linkdin
*RSS
*Youtube
*Shareit

You can also choose different styles of icons you want to use.

== Installation ==

1. Upload social-icons-by-demoify to the /wp-content/plugins/ directory
2. Activate social-icons-by-demoify through the 'Plugins' menu in    WordPress
3. Go to widgets and add social-icons-b.-demoify

== Frequently Asked Questions ==

=What are plugins?=
Plugin is a piece of software containing several functions that makes a user use its functionality as per requirement.

= Does this plugin work with newest WP version and also older versions? =
Yes, this plugin works really fine with Version 4.5.3. So you always should run the latest WordPress version for a lot of reasons.

=How do I add links to social icons?=

**Manual**
You can add link to the choosen social icons in a simple way
->Go to widgets
->Choose social-icons-by-demoify
->Insert necessary links
